Captured with Canon 50D using an EF70-200mm f/4, ISO 100, 1/50s.  The Java sparrow has been a popular cage bird in Asia for centuries, first in China's Ming Dynasty and then in Japan from the 17th century, frequently appearing in Japanese paintings and prints.
