<h1 class="toph">About</h1>

Just a place to track books I've read / am reading built with eleventy (11ty), some duct tape, javascript, markdown, an assist from GitHub to host it, and GitHub Copilot (Claude) to get my 11ty filters working the way I want.

The **HKBC** in the nav stands for Hong Kong Book Club, which is a subset of the **Reading** list. 

There is a list of [authors](/writers) mentioned on this site, too.

If the world ever begins to return to normalcy, I may even have a [blog](/blog).
