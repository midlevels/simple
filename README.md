# simple

This repository is a text-only version of `startmeup`, preserving the same Eleventy, CMS, CSS, and Nunjucks structure while suppressing rendered media output. 

**NOTE:** There are two npm install commands, one for the main 11ty static site builder, and another for the CMS. 

## Common npm scripts

| Command | What it does |
|---------|-------------|
| `npm start` | Local dev server with live reload |
| `npm start cms` | Launches cms on localhost:3000 |
| `npm run build:all` | Build + generate Pagefind search index |
